The shared framework

What every persona was measured on

The six addictiveness dimensions are persona-agnostic — they target specific cognitive and physiological mechanisms of compulsive re-listening that work the same way regardless of the artist's tessitura, genre, or audience.

1

Hook Strength

Hook timing (0:08–0:14), placement in the artist's resonant chest-belt range, repetition density (4–5×/chorus, ≤8 syllables). The phrase the listener hums involuntarily hours later.

2

Groove

BPM sweet spot (typically 86–140 depending on persona), kick pattern, snare/clap snap, bassline pocket, swing %. The body responds before the mind decides.

3

Replay Value

Ear-candy density per track (2–3 small sonic surprises with explicit timestamps). Each replay reveals a new detail — the song never feels exhausted.

4

Dynamic Payoff

Strip-back → swell → release tension map per track. The "held breath → exhale" at the 2:14–2:42 bridge → final-chorus window. Dopaminergic tension-release wired into bridge architecture.

5

Loop-ability

Per-track outros pull into next track. Album T12 closing loops near-seamlessly back to T1. Removes the natural stopping point that triggers playlist-switch.

6

Emotional Pull

Core feeling per track (seen / lifted / energized / understood / called / pastored / Spirit-fell). Listeners return for the feeling, not just the sound. Dominant dimension across all seven rated personas.
